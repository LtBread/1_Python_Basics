    Строки 8-9

    if os.path.exists(templates_path):
    shutil.rmtree(templates_path)

    удаляю дерево templates, если, оно уже есть.
    Так обновляется дирректория шаблонов.
    Исходил из логики, что если в директории есть файл index.html,
    то это нужная директория, и что там только файлы расширения .html,
    то есть ничего лишнего там нет. Поэтому создаём директорию с таким же именем
    и копируем туда все файлы из исходной директории.
    Таким образом, отражается текущее состояние корневой директории,
    если появятся новые директрии с .html, то они попадут в templates.
    Если какие-то директории будут удалены, то из templates они тоже
    удалятся.

    В 14й строке можно было написать (номер строки может измениться, могу не уследить)

    if os.path.exists(os.path.join(root, search_file)):
        ...

    Но условие

    if search_file in set(files):
        ...

    проще и наглядней, сложность О(1), думаю это лучше.
